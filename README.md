# Proyecto de Reconocimiento de Vehículos

Este proyecto implementa un sistema básico de **detección de vehículos en video** utilizando la librería **OpenCV** en Python y un clasificador Haar Cascade pre-entrenado.  
El objetivo es mostrar visualmente cómo el sistema reconoce autos y motos en movimiento mediante rectángulos y una línea guía.

 Archivos incluidos
- `reconocimientoVehiculos.py` → Script principal en Python para la detección de vehículos.  
- `requirements.txt` → Archivo con las dependencias necesarias para ejecutar el proyecto.  
- `cars.xml` → Modelo Haar Cascade utilizado como clasificador pre-entrenado para la detección de autos.  
- `videoPrueba1.mov` → Video de prueba (opcional, solo si se requiere en la entrega).
---
Para ejecutar proyecto 
python reconocimientoVehiculos.py
